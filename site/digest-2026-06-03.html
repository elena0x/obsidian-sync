Peter Yang (Product at Roblox)

https://x.com/petergyang/status/2041331383344443795
Roblox 产品经理 Peter Yang 分享了与 a16z 合伙人 Anish Acharya 关于 AI agent 时代工作未来的对话要点。他认为 coding agent 将吞噬所有知识型工作——他现在 80% 的文档、幻灯片和数据分析都先交给 AI 完成，再手动润色。2-3 人的小团队配上 agent 集群将击败臃肿的大组织。功能性应用会萎缩，娱乐性应用则会留存。最触动他的是，他的 OpenClaw agent 对他说："你的孩子才 7 岁和 4 岁，他们很快就会长大——多花时间陪他们。"他还反驳了用 token 消耗量衡量生产力的做法，称其"跟用代码行数衡量一样愚蠢"。

Aaron Levie (CEO at Box)

https://x.com/levie/status/2041347596342460439
Box CEO Aaron Levie 发表长文解释为什么 AI agent 不会消除工作，而是改变工作。当 agent 替你干活时，你的工作上升了一层抽象：你要想清楚让 agent 做什么、给正确的指令、提供上下文、在跑偏时干预、审核最终产出、并将其整合到更大的工作中。"你现在成了编辑、经理、制片人。品味很重要，理解手艺很重要，技能最终仍然重要。"他认为 AI 自动化的是烦人的部分，留下的是有趣的部分——但工作还在，只是形式不同了。
